## Resumo

Primeiro release oficial do **FRV-OMIE Dashboard de Vendas** — sistema web que substitui planilha Excel de controle de vendas com ~R$200k/mês de faturamento. O sistema inclui integração completa com o Omie CRM via Edge Functions, RBAC por roles, design system premium V2.0 e migração automatizada de dados históricos.

## Highlights Técnicos

### Integração Omie
- **3018 clientes** disponíveis via API Omie
- **6702 contas receber** sincronizáveis
- **10263 pedidos** indexados
- Sync sequencial no frontend (vendedores → clientes → vendas) para contornar timeout de 60s
- Reconciliação de vendedores por nome (match entre seeds e dados Omie)

### Performance
- Code splitting React.lazy + Suspense em todas as rotas
- manualChunks Vite: vendor-react, vendor-ui, vendor-charts, vendor-query
- 1300KB → 435KB (67% de redução)

### Segurança
- RLS policies por role (admin/gerente/vendedor)
- JWT via Supabase Auth
- ProtectedRoute + RoleGuard em todas as rotas protegidas
- Edge Functions isoladas com service_role key

### Design System V2.0
- Paleta Omie Blue & Teal com CSS variables (light/dark)
- Tipografia tri-fonte: Space Grotesk + Noto Sans + JetBrains Mono
- KPI Cards com gradient icons e micro-animações
- Sidebar com gradient nav active
